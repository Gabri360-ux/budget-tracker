# Budget Tracker - Calculadora de Presupuesto Personal

![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)

Aplicación web para llevar el control de tus ingresos y gastos personales. Los datos se guardan en el navegador con localStorage, así que no se pierden al recargar la página.

## Características

- Registrar transacciones con descripción, monto y tipo (ingreso o gasto)
- Cálculo automático de ingresos totales, gastos totales y balance
- Historial de transacciones con opción de eliminar cada registro
- Datos persistentes gracias a localStorage
- Diseño moderno con animaciones y responsive

## Tecnologías usadas

- HTML5
- CSS3 (Grid, Flexbox, animaciones)
- JavaScript (ES6+)
- LocalStorage API
